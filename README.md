# Library Management System

A full-stack web application for managing a library system, built with **Spring Boot** and **React**.

##  Features

- 🔐 JWT Authentication (Login/Register)
- 👤 Role-based Authorization (Admin/User)
- 📚 Book Management (CRUD)
- 📦 Borrow / Return System
- 📊 Admin Dashboard (analytics & statistics)
- 🔑 Forgot / Reset Password (Email-based)

##  Technologies

### Backend
- Java
- Spring Boot
- Spring Security (JWT)
- Hibernate / JPA

### Frontend
- React.js
- Axios
- Chart.js

### Database
- PostgreSQL / MySQL

### Others
- Docker (optional)
- Git & GitHub
